# Taiwan-Central-Bank-Macro-Analysis
Description: Quantitative analysis and visualization of Taiwan’s Central Bank (CBC) asset trends (1998-2024) using R. Investigating the correlation between FX policy and financial sector innovation.

# Analysis of Taiwan's Central Bank Assets (1988-2024)

## Project Overview
This repository contains an R-based analysis of the Central Bank of Taiwan's (CBC) balance sheet expansion. 

## Key Insights
* **The "National Hedger" Strategy:** Visualizing the CBC's assets-to-GDP ratio.
* **Financial Innovation:** Reflecting on observations from my **Yuanta Securities IB internship** regarding the trade-offs of a high-stability FX policy.

## Technical Skills Demonstrated
* **R Programming:** Data cleaning and reshaping with `tidyr`.
* **Data Visualization:** Advanced plotting with `ggplot2` (custom themes, dual-axis logic).
* **Economic Analysis:** Connecting macro data to financial sector growth.
